python numpy的一些tricks

Sep 18, 2018 | tricks | 阅读

第一次写博客，把这两天新学的tricks记录下来，以便翻阅查找。

python 的一些技巧：

  • del a, b：删除某个变量名的内存空间，防止内存错误。

  • plt.plot(x, y, '-o')

  • plt.subplot(x, y, i)：x长，y宽，i第几幅画，从1开始数。

  • lambda 表达式，匿名函数

    add = lambda x, y: x + y

    add(1, 2) => 3

numpy 里面的一些函数：

  • axis = 0：以列为计算对象；axis = 1：以行为计算对象（这点和平时用法相反）。

    axis = 0 对横轴进行操作，在运算过程中其运算方向表现为纵向运算

    axis = 1 对纵轴进行操作，在运算过程中其运算方向表现为横向运算

  • numpy数组里面的()和[]不要搞混了，取索引是[i, j]

  • sum(matrix, axis = 1)，对每行进行相加，得到一个向量，默认返回行向量。

  • np.linalg.norm(x, ord = None, ...)，求向量x的范数（是一个标量）。

    ord参数的用法：

    默认ord = 2，L2范数
    $$
    \sqrt{\sum_i x_i ^ 2}
    $$
    ​ ord = 1，L1范数
    $$
    \sum_i{|x_i|}
    $$
    ​ ord = np.inf，L无穷范数
    $$
    \max(|x_i|)
    $$

  • np.flatnonzero(y_train == y)：输出y_train == y时的索引数组（默认为非零元素的索引数组）。

  • np.random.choice(idx, sample_per_class, replace = False, ...)：从idx中随机选取sample_per_class个，无放回；replace = True 有放回。

  • a = np.reshape(a, (2, -1))： -1的意思是自动计算相应维度。

  • np.diag(x)：

    1. x 是矩阵，返回对角元素向量。
    2. x 是向量，返回对角阵。

  • np.repeat(a, reqeats, axis = None)

    1. axis = None 每个元素复制repeats次，返回行向量。[1, 1, 2, 2, ……]

      1. axis = 0 沿y轴复制 [[1, 2],

        ​ [1, 2],

        ​ [3, 4],

        ​ [3, 4]]

        1. axis = 1 沿x轴复制 [[1, 1, 2, 2], 

          [3, 3, 4, 4]]

  • np.array_split(a, num_folds)：平分数组a

  • np.dot(a, b) 矩阵相乘 np.multiply(a, b)矩阵元素相乘。

  • 求精度可用np.mean(a = b), np.mean(y_train == y_train_pred)。

  • np.random.seed(n)：接下来n次随机生成的数（组）都相同。

  • np.random.rand(d0, d1) ：随机生成0到1之间的任意一数， d0和d1是维度。

  • np.random.randn(d0, d1)：生成的数组服从正态分布

  • np.maximum(x, y) or np.maximum(a, 0)：x，y逐位比较取较大者

    =>[0, 0, 1, 2, 0]
