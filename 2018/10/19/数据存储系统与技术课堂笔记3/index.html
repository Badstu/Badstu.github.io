Cache设计问题

  • Cache数据结构
    • Cache里的数据组织结构，如队列、数组等
    • 数据进入Cache的映射方式等
  • Cache内容管理
    • 哪些数据要进行缓存、哪些数据可不缓存
    • Cache满了怎么处理
  • Cache一致性管理
    • 缓存与主存数的一致性管理
    • 多个缓存间数据一致性管理

Cache数据结构-处理器高速缓存

  • Cache line
    • Cache Metadata
    • Cache Data
  • Cache 映射方式
    • 直接映射：（块地址）MOD（Cache中的块数）
    • 组映射：（块地址）MOD（Cache中的组数）
    • 全映射：一个块可以放到Cache中的任何位置

Cache的内容管理-处理器高速缓存

  • 内容管理确定哪些数据块需要在Cache中缓存，理想状态下Cache中缓存未来一段时间内需要访问的数据块
  • 当Cache中数据块满了之后，心需要注入的数据块如何替换已有的数据块
    • 随机替换策略
    • 最近最少使用替换策略（LRU）
    • 先进先出替换策略（FIFO）

Cache一致性管理-处理器高速缓存

  • 由于Cache的存在，数据可能存在于Cache和/或下一级存储层次中，当数据写入时
    • 写命中 + 写未命中
      • 写直达 + 不写分配
      • 写回法 + 写分配法
