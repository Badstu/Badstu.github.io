科技论文写作课堂笔记2

Oct 15, 2018 | 课堂笔记 | 阅读

时间：10月15日上午 周一

科研论文就是八股文，起承转合，论述等等，按照框架和体系书写，很容易与他人沟通

Main part

  • Algorithm
  • System design
  • Enhancement/optimization
  • Analysis
  • Experiments
  • 举例子的作用是让别人理解你的算法
  • 理论分析包括正确性分析、复杂度分析、最差情况分析和平均时间分析
  • 真实数据的实验
  • 算法介绍包含两个部分，静态部分：数据结构；动态部分：算法怎么运行

More papers

  • Access path selection in a relation database management system. SIGMOD, 1979.
  • ARIES: A Transaction Recovery Method Supporting Fine-Granularity Locking and Partial Rollbacks Using Write-Ahead Logging. ACM Transaction on Database Systems, 1992.
  • The R*-Tree: An Efficient and Robust Access Method for points and Rectangles. SIGMOD Conference 1990.
  • Fast Algorithms for Mining Association Rules in Large Databases. VLDB 1994.

能用文字去表达的就用文字去表达，需要强调的地方用图，用表，用伪代码

用1/4页到半页之间讲清楚running example（同时涵盖很多例子）
